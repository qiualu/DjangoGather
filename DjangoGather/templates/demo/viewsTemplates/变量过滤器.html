普通 变量 : {{ putong }}

普通 变量 : {{ leghtStr }}

过滤器1 变量 : {{ leghtStr|truncatewords:"5" }} ---> 这个将显示变量 bio 的前5个词。

过滤器2 变量 : {{ leghtStr|addslashes }} ---> 添加反斜杠到任何反斜杠、单引号或者双引号前面。

过滤器3 变量 : {{ leghtStr|truncatechars:"5" }} ---> 555 如果字符串包含的字符总个数多于指定的字符数量，那么会被截断掉后面的部分。

default
default 为变量提供一个默认值。
如果 views 传的变量的布尔值是 false，则使用指定的默认值。
以下值为 false：
0 0.0 False 0j "" [] () set() {} None

\{{ name|default:"菜鸟教程666" }\}

forloop.counter: 顺序获取循环序号，从 1 开始计算
forloop.counter0: 顺序获取循环序号，从 0 开始计算

forloop.revcounter0: 倒叙获取循环序号，结尾序号为 0
